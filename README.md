# ValAnimation Web ❤️

A Valentine's Day web app with floating hearts, cycling photos, romantic message, and background music. Mobile-first & deployable on GitHub Pages.

## Local Development

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

1. Push to the `main` branch.
2. Go to repo **Settings → Pages → Source** and select **GitHub Actions**.
3. The workflow builds and deploys automatically on every push.
4. Your site will be at `https://<username>.github.io/<repo-name>/`
